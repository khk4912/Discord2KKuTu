## Discord2KKuTu에 오신 것을 환영합니다!

Discord2KKuTu는 채팅 프로그램 Discord와 웹 끝말잇기 게임인 KKuTu의 로비채팅을 서로 연동시켜주는 프로젝트입니다.
사용자들은 지정된 채널에서 끄투 로비와 특정한 디스코드 채널에서 서로 소통할 수 있습니다.
